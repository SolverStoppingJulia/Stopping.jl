369 """
370 broadcast(f, As...)
371
372 Broadcasts the arrays, tuples, `Ref`s, nullables, and/or scalars `As` to a
373 container of the appropriate type and dimensions. In this context, anything
374 that is not a subtype of `AbstractArray`, `Ref` (except for `Ptr`s), `Tuple`,
375 or `Nullable` is considered a scalar. The resulting container is established by
376 the following rules:
377
378 - If all the arguments are scalars, it returns a scalar.
379 - If the arguments are tuples and zero or more scalars, it returns a tuple.
380 - If the arguments contain at least one array or `Ref`, it returns an array
381 (expanding singleton dimensions), and treats `Ref`s as 0-dimensional arrays,
382 and tuples as 1-dimensional arrays.
383
384 The following additional rule applies to `Nullable` arguments: If there is at
385 least one `Nullable`, and all the arguments are scalars or `Nullable`, it
386 returns a `Nullable` treating `Nullable`s as "containers".
387
388 A special syntax exists for broadcasting: `f.(args...)` is equivalent to
389 `broadcast(f, args...)`, and nested `f.(g.(args...))` calls are fused into a
390 single broadcast loop.
