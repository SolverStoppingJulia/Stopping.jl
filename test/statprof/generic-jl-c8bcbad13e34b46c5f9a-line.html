542 """
543 norm(A::AbstractMatrix, p::Real=2)
544
545 For matrices, the matrix norm induced by the vector `p`-norm is used, where valid values of
546 `p` are `1`, `2`, or `Inf`. (Note that for sparse matrices, `p=2` is currently not
547 implemented.) Use [`vecnorm`](@ref) to compute the Frobenius norm.
548
549 When `p=1`, the matrix norm is the maximum absolute column sum of `A`:
550 ```math
551 \\|A\\|_1 = \\max_{1 â¤ j â¤ n} \\sum_{i=1}^m | a_{ij} |
552 ```
553 with ``a_{ij}`` the entries of ``A``, and ``m`` and ``n`` its dimensions.
554
555 When `p=2`, the matrix norm is the spectral norm, equal to the largest
556 singular value of `A`.
557
558 When `p=Inf`, the matrix norm is the maximum absolute row sum of `A`:
559 ```math
560 \\|A\\|_\\infty = \\max_{1 â¤ i â¤ m} \\sum _{j=1}^n | a_{ij} |
561 ```
562
563 # Example
564
565 ```jldoctest
566 julia> A = [1 -2 -3; 2 3 -1]
567 2Ã3 Array{Int64,2}:
568 1 -2 -3
569 2 3 -1
570
571 julia> norm(A, Inf)
572 6.0
573 ```
574 """
575 function norm(A::AbstractMatrix, p::Real=2)
576 if p == 2
577 return norm2(A)
578 elseif p == 1
579 return norm1(A)
580 elseif p == Inf
581 return normInf(A)
582 else
583 throw(ArgumentError("invalid p-norm p=$p. Valid: 1, 2, Inf"))
584 end
585 end

644 function vecdot(x, y) # arbitrary iterables
645 ix = start(x)
646 if done(x, ix)
647 if !isempty(y)
648 throw(DimensionMismatch("x and y are of different lengths!"))
649 end
650 return dot(zero(eltype(x)), zero(eltype(y)))
651 end
652 iy = start(y)
653 if done(y, iy)
654 throw(DimensionMismatch("x and y are of different lengths!"))
655 end
656 (vx, ix) = next(x, ix)
657 (vy, iy) = next(y, iy)
658 s = dot(vx, vy)
659 while !done(x, ix)
660 if done(y, iy)
661 throw(DimensionMismatch("x and y are of different lengths!"))
662 end
663 (vx, ix) = next(x, ix)
664 (vy, iy) = next(y, iy)
665 s += dot(vx, vy)
666 end
667 if !done(y, iy)
668 throw(DimensionMismatch("x and y are of different lengths!"))

793 such that `A*X == B` when `A` is square. The solver that is used depends upon the structure
794 of `A`. If `A` is upper or lower triangular (or diagonal), no factorization of `A` is
