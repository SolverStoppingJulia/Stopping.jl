68 """
69 dlopen(libfile::AbstractString [, flags::Integer])
70
71 Load a shared library, returning an opaque handle.
72
73 The extension given by the constant `dlext` (`.so`, `.dll`, or `.dylib`)
74 can be omitted from the `libfile` string, as it is automatically appended
75 if needed. If `libfile` is not an absolute path name, then the paths
76 in the array `DL_LOAD_PATH` are searched for `libfile`, followed by the
77 system load path.
78
79 The optional flags argument is a bitwise-or of zero or more of `RTLD_LOCAL`, `RTLD_GLOBAL`,
80 `RTLD_LAZY`, `RTLD_NOW`, `RTLD_NODELETE`, `RTLD_NOLOAD`, `RTLD_DEEPBIND`, and `RTLD_FIRST`.
81 These are converted to the corresponding flags of the POSIX (and/or GNU libc and/or MacOS)
82 dlopen command, if possible, or are ignored if the specified functionality is not available
83 on the current platform. The default flags are platform specific. On MacOS the default
84 `dlopen` flags are `RTLD_LAZY|RTLD_DEEPBIND|RTLD_GLOBAL` while on other platforms the
85 defaults are `RTLD_LAZY|RTLD_DEEPBIND|RTLD_LOCAL`. An important usage of these flags is to
86 specify non default behavior for when the dynamic library loader binds library references to
87 exported symbols and if the bound references are put into process local or global scope. For
88 instance `RTLD_LAZY|RTLD_DEEPBIND|RTLD_GLOBAL` allows the library's symbols to be available
89 for usage in other shared libraries, addressing situations where there are dependencies
90 between shared libraries.
91 """

122 """
123 find_library(names, locations)
124
125 Searches for the first library in `names` in the paths in the `locations` list,
126 `DL_LOAD_PATH`, or system library paths (in that order) which can successfully be dlopen'd.
127 On success, the return value will be one of the names (potentially prefixed by one of the
128 paths in locations). This string can be assigned to a `global const` and used as the library
129 name in future `ccall`'s. On failure, it returns the empty string.
130 """
131 function find_library(libnames, extrapaths=String[])
132 for lib in libnames
133 for path in extrapaths
134 l = joinpath(path, lib)
135 p = dlopen_e(l, RTLD_LAZY)
136 if p != C_NULL
137 dlclose(p)
138 return l
139 end
140 end
141 p = dlopen_e(lib, RTLD_LAZY)
142 if p != C_NULL
143 dlclose(p)
144 return lib
145 end
146 end
147 return ""
148 end
